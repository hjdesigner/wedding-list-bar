# Wedding Bar List

## Structure of the project
- JS : js/main.js
- SASS : sass/stylus.scss
- CSS minified : css/stylus.css  
- IMAGES : images/
- HTML : index.html

## Libs
- Carousel : Slick
- Scroll : mCustomScrollbar

## Dependency to run the SASS
- Node.js => V6
- Yarn
- Gulp

## SASS Build command
- yarn install
- yarn run sass
